Overview

An infographic is a visual representation of complex data. Infographics allow users to view graphics and words to understand new information. However, infographics are fundamentally not accessible. Typical infographics are images, which are created in Word, Illustrator, or using free web infographic creators. Regardless of how they are created, infographics are known for these design features:

  • Use pictures to represent words
  • Group information by colors
  • Cluster information in a static order
  • Use extreme color contrast

If accessibility is not considered while designing an infographic, the content may be unrecognizable to screen readers. The following resource provides an overview of creating image and HTML/CSS infographics.

 Creating Accessible Infographics

Beginner Infographic: Image

The majority of infographics are images created using software or web applications. The most efficient way to create an accessible infographic (from an image) is to include alternative text. Alternative text (alt text) is a typed explanation of the content of an image and allows a screen reader to access the information. Alt text ensures the infographic is accessible to all users. For infographics, alt text is written from top to bottom and left to right. If the infographic is placed on the web, there are two ways to display the alt text:

  1. Display the full typed description (alt text) directly under the infographic. Scroll down, directly below the infographic on the WebAIM site, to view an example.
  2. Create a separate page with the alt text and link to the page directly under the infographic. 

A study conducted by WebAIM (2015), surveyed over 2,500 participants about their use of screen readers in digital spaces. The survey highlights a significant (43% of participants) preference for complex images to be presented by using optional text made available on the same page. It's valuable to know the preferences of the intended audience. For additional support about how to write alternative text, please review Best Practices for Accessible Images.

Advanced Infographic: HTML and CSS

Advanced developers and designers create infographics using HTML and CSS. If the HTML and CSS are developed using custom codes to include accessibility, then the infographic becomes accessible to screen readers. In addition, an HTML infographic increases Search Engine Optimization (SEO) whereas an infographic that is an image does not. For additional support about how to write alternative text, please review Best Practices for Accessible Images.
