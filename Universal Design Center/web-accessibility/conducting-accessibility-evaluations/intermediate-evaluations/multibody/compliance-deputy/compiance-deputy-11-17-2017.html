Compliance deputy is a newly developed browser-based solution that allows developers and content creators to evaluate and repair accessibility errors before publishing them. Compliance Deputy was developed by Cryptzone in collaboration with the CSU ATI and the Universal Design Center. It measures compliance using the CSU ATI Accessibility Requirements (DOCX) and the UDC Web Accessibility Criteria. It is supported by browsers IE 9.0 and above, Chrome 40.0 and above and Safari 7.0 and above.

To download Compliance Deputy, please visit UDC's Testing and Learning Tools page.
