Automated Testing Tools

WebAIM WAVE tool

Activate the WAVE toolbar and navigate to the “Details” panel. In that panel navigate to the “Features” section and manually review all the image maps’ alternative text that are present in the webpage.

The following accessibility errors related to image maps are found by wave:

    • Image map area missing alternative text 
    • Alternative text is not present for an image that has hotspots. 
    • Alternative text is not present for an image map area (hotspot).
    • Server-side image map is present 

Compliance Sheriff

To test for image maps in Compliance Sheriff, the following checkpoint groups are used: Section F for client-side image maps and Section E for server-side image maps.  For more information regarding the checkpoints visit the CSU ATI Accessibility Requirements page.
