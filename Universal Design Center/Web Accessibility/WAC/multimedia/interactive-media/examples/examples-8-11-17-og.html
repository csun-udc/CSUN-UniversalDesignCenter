Transcripts

The screenshot below is of a video from the CaptionSync website that portrays the use of transcripts. This example demonstrates the two methods transcripts can be incorporated; as interactive transcripts or automated transcripts.

Interactive transcripts are synchronized to a multimedia file. As the multimedia file is played, the interactive transcripts will usually highlight in association with the file. Interactive transcripts are usually selectable and allow users to move to specific places within the multimedia file. Automated transcripts are a written summary of the audio within the video. They are usually displayed below the multimedia file.
