<h3><a id="a1"></a>What is Alternative or Alt Text?</h3>
<p><strong>Alternative Text</strong>&nbsp;or <strong>Alt text&nbsp;</strong>is a text replacement for an image and is used to explain what the image is and what the image is trying to convey to the user. It requires adding the <code>alt </code>attribute to an image (<code>img</code>) element.</p>
<p style="text-align: center;"><b>Syntax:</b><code><strong><span style="color: #c00000;">&lt;img </span> <span style="color: #105168;"> alt=</span></strong><strong><i><span style="color: #6600cc;">"text"</span></i></strong></code><span style="color: #c00000;">&gt;</span></p>
<a href="#beginning1"><div class="top"> back to top </div></a>
<br>
<h3><a id="a2"></a>Why do I need alt text?</h3>
<p>Including alt text in your images ensures that your image is accessible to people with disabilities. The primary purpose of alt text is to allow screen reader users, such as individuals with cognitive or visual disabilities, to obtain a text interpretation of the image to allow them to understand the importance of the image. &nbsp;Moreover, alt text can all be beneficial for all users in that it can serve as an image replacement if the image cannot be displayed on the page and it can be used to help search engines identify your website more easily.&nbsp;</p>
<a href="#beginning1"><div class="top"> back to top </div></a>
<br>
<h3><a id="a3"></a>How do you know if an image needs alt text?</h3>
<p>An image requires alt text if it conveys important information. The following do not need alt text:</p>
<ul>
<ul>
<li>Images that serve for <b>decoration only</b>. Decorative images are considered images that are used for layout and do not present important content. If a decorative image is present, the alt text attribute should be null (<code>Alt=””</code>).</li>
<li>Images that have a <b>caption</b> do not need alt text. Captions are text descriptions for an image that are displayed on the page, thus allowing sighted and non-sighted users information about the image. &nbsp;If a caption is present, the alt text attribute should be null (<code>Alt=””</code>).</li>
<li>Images with the <b>text description within the content of the page</b> do not require alt text. If the image description is within the content of the page, the alt text attribute should be null (<code>Alt=””</code>).</li>
</ul>
</ul>
<p><b>NOTE:</b> For images that do not need alt text it is recommended that the alt text attribute is null (<code>Alt=””</code>). This will ensure that assistive technologies skip over the image and that no accessibility-testing applications &nbsp;signal an error if the alt text is not provided for these images.&nbsp;</p>

<h3><a id="a4"></a>How do I write great Alt Text?</h3>
<p>Writing great alt text takes time and practice. There is no one way to do it. Nonetheless, there are tips to follow to write good alt text:</p>
<ul>
<ul>
<li><b>Be precise</b>. Alt text should accurately present the same content and function as is presented in the image.</li>
<li><b>Be brief</b>. Alt text of an image should be short and to the point. Listening to long alt text can be burdensome for assistive technology users.&nbsp;Longer alt text might also be a problem because it may provide more information to assistive technology users&nbsp;giving a disadvantaged to sighted users.&nbsp;</li>
<li><b>DO NOT repeat information</b>. Alt text should not provide the exact same information as text within the context of the image. Moreover, the same alt text should not be used different images.</li>
<li><b>DO NOT use the phrases "image of ..." or "graphic of ..." to describe the image.</b> Typically, assistive technologies will announce to a user the presence of an image. Therefore, including phrases such as “image of…”, “graph of…” causes unnecessary redundancy for users.</li>
</ul>
</ul>
