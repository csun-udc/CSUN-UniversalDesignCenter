<h3 id = "beginning3">Manual Testing</h3>
<p><strong>Page title</strong>: Open the website on your web browser. View the title page and verify that the page title is descriptive and accurately describes the page. Then review the source code to verify the title tag.</p>
<p><strong>Resize text</strong>: To resize your text on your browser download browser extensions or use your browser features.</p>
<ul>
<li><strong>Firefox</strong>: To resize your text up to 200% use the Firefox “Zoom Text Only” feature.</li>
<li><strong>Google Chrome</strong>: Using the “Zoom Text Only” add on will allow you to increase the text size of the page. For further information on how to install the toolbar, please refer to our&nbsp;<a href="https://w1dev.csun.edu/node/116206/checking-accessibility-your-documents">Document Accessibility Tools</a>&nbsp; page</li>
</ul>
<h3>Automated Testing Tools</h3>
<h4 style="padding-left: 30px;"><a href="https://chrispederick.com/work/web-developer/">Web Developer Toolbar</a></h4>
<ul>
<ul>
<li><strong>Headings</strong>:&nbsp;Using the Web Accessibility Toolbar, navigate to the website on your web browser. Activate the Web accessibility toolbar, then select <strong>Structure</strong> menu and select <strong>Headings</strong>, then verify that headers are present and organized correctly. &nbsp;</li>
</ul>
</ul>
<h4 style="padding-left: 30px;"><a title="http://wave.webaim.org/" href="http://wave.webaim.org/" target="_self">WebAIM WAVE tool</a></h4>
<ul>
<ul>
<li><strong>Headings</strong>:&nbsp;To view the headings using the Wave Toolbar, activate the toolbar. Then in the Styles page, navigate to <strong>Outline</strong> to view the heading structure of the page.</li>
<li><strong>Lists</strong>: Using the Web Accessibility Toolbar, navigate to the website on your web browser. Select Structure menu and select <strong>List Items</strong>, then verify if the all list items have the HTMl list element.</li>
<li><strong>Ordered lists</strong>:&nbsp;Should have the &lt;ol&gt; tag and unordered lists should have the &lt;ul&gt; tag.</li>
</ul>
</ul>
<h4 style="padding-left: 30px;"><a href="https://www.csun.edu/universal-design-center/compliance-sheriff">Compliance Sheriff</a></h4>
<p style="padding-left: 30px;">In Compliance Sheriff, the checkpoint group &nbsp;<strong>CSU ATI Semantic Requirements </strong>will review all semantic and header structures.&nbsp; For more information regarding the checkpoints visit the <a href="https://www.csun.edu/universal-design-center/compliance-sheriff-checkpoints"><span style="text-decoration: underline;">CSU ATI Accessibility Requirements</span></a> page.&nbsp;</p>
