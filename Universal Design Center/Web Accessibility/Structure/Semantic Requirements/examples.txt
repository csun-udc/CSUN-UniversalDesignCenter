<h3 id = "beginning2">Page titles:</h3>
<h4>&nbsp; &nbsp; &nbsp; &nbsp; Descriptive page title</h4>
<p style="padding-left: 60px;">"<em>Company History</em>"</p>
<p style="padding-left: 60px;">The descriptive page title give<br />s user's information about the content that will be presented on the page. The user will not what kind of information to expect based on the page title.</p>
<h4>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Non-descriptive page title</h4>
<p style="padding-left: 60px;"><em>"Learn More"</em></p>
<p style="padding-left: 60px;">The non-descriptive page title does not give users any information about the content of the page. The phrase "learn more" is vague and can refer to any topic on the website. For example, if a website about growing crops contains a page with the title "Learn More." Users will not know what content the page "Learn More" will contain. A descriptive page title would be "Farming Techniques." That page title is descriptive and summarizes the content on the page.</p>

<h3 style="text-align: left;"><b>Lists:</b></h3>
<p style="padding-left: 30px;">An <strong>unordered list</strong> &lt;ul&gt;usually consistent of bullet points. For example:</p>
<ul style="padding-left: 30px;">
<ul>
<li>CSU Northridge</li>
<li>CSU Fullerton</li>
<li>CSU Long Beach</li>
<li>CSU San Diego</li>
</ul>
</ul>
<p style="padding-left: 30px;"><strong>Ordered lists</strong> &lt;ol&gt; are organized in a descending order with the use of 1, 2, 3 or a, b, c. For example:</p>
<ol style="padding-left: 30px;"><ol>
<li>Milk</li>
<li>Eggs</li>
<li>Cereal</li>
<li>Bread</li>
</ol></ol>
