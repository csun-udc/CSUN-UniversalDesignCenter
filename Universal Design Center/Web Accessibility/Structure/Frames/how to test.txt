<h3 id = "beginning2">Manual Testing</h3>
<p style="padding-left: 30px;">Review the web pages' source code to view the frame’s title and verify if the title is descriptive. For example, a non-descriptive title for a frame is “Frame 1”. A descriptive frame title would be “Go Matadors Facebook Feed”&nbsp;</p>
<h3>Automated Testing Tools</h3>
<h4 style="padding-left: 30px;"><a href="https://chrispederick.com/work/web-developer/">Web Developer Toolbar</a></h4>
<p style="padding-left: 30px;">Using the Web Accessibility Toolbar, navigate to the website on your web browser. Select <strong>Outline</strong> menu and select <strong>Outline Frames</strong>, this will then highlight all frames present on the webpage.</p>

<h4 style="padding-left: 30px;"><a href="http://wave.webaim.org/">WebAim WAVE Toolbar</a></h4>
<p style="padding-left: 30px;">Activate the WAVE toolbar and navigate to the Styles page. In the Details section, navigate to Structural Elements then verify if any frames are present in the page.&nbsp;</p>
<p style="padding-left: 30px;">The following are accessibility errors and Structural &amp; Semantic elements found by WAVE:&nbsp;</p>
<p style="padding-left: 60px;">[[{"type":"media","view_mode":"media_large","fid":"532771","attributes":{"alt":"Frame missing title error","border":"0","class":"media-image","height":"20","typeof":"foaf:Image","width":"20"}}]]<strong>:&nbsp;</strong>A frame does not have a "title" attribute or value.</p>
<p style="padding-left: 60px;">[[{"type":"media","view_mode":"media_large","fid":"532746","attributes":{"alt":"Inline Frame","border":"0","class":"media-image","height":"25","typeof":"foaf:Image","width":"25"}}]]<b>:&nbsp;</b>An inline frame (iframe) is present<b>.</b></p>
<h4 style="padding-left: 30px;">Chrome Ad-on</h4>
<p style="padding-left: 30px;">To remove iframes in a page use the chrome ad-on: <a href="https://chrome.google.com/webstore/detail/remove-iframe/pbaicmldjijbgdadgepfemcolbpbdfai?hl=en">Remove iFrame</a></p>
<h4 style="padding-left: 30px;"><a href="https://www.csun.edu/universal-design-center/compliance-sheriff">Compliance Sheriff</a></h4>
<p style="padding-left: 30px;">To test for frames and iframes in Compliance Sheriff the checkpoint is Section I. For more information regarding the checkpoints visit the <a href="https://www.csun.edu/universal-design-center/compliance-sheriff-checkpoints">CSU ATI Accessibility Requirements</a> page.&nbsp;</p>
