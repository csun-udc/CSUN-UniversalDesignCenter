Yes for Accessibility!

Interact with the 10 point checklist while reviewing a newly converted Word document. The links provide a guide to complete each checkpoint. Finally, click, "Yes" as each checkpoint is complete.

  1. Are styles used for title and section headers?  
  2. Are styles  used instead of selecting font size and effects?  
  3. Are numbered lists and bullets used (when appropriate)?  
  4. Do images and objectsspan> have meaningful alternative text (alt text)?  
  5. Was color contrast considered  
  6. Are tables used to cross-reference information?  
  7. Are links active and use meaningful link text?  
  8. Are columns used to condense large amounts of information?  
  9. Has the Check Accessibility tool been run?  
  10. Has the document been converted to a PDF?  

Additional Considerations

These remaining items provide additional accessibility consideration for Word Accessibility.

  • Use a concise document file name that doesn't include spaces i.e. Word-Accessibility-Example.docx
  • A table of content is provided for longer documents (can be automatically created by using proper headers)
  • A document title and author are present
  • All comments are removed or have been turned off
